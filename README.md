# 💵 Dollar Gravity Framework

**Einstein Fabric Planetary Visual** — Interactive 3D model of global USD gravity.

Live: [https://powder-ranger.github.io/dollar-gravity-framework/](https://powder-ranger.github.io/dollar-gravity-framework/)

Each sovereign entity = planet/moon. Colored gravity fields visualize money flows (gold), military power (red halos), alliances (blue arcs). Zoom, orbit, click for details.
